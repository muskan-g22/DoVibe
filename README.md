# DoVibe 🎵

DoVibe is an Android application designed to provide users with a smooth and engaging music experience. The app offers a clean UI, easy navigation, and interactive features for enjoying music anytime.

## ✨ Features

* 🎶 Music Player Interface
* 🔍 Search Functionality
* ⏯ Play / Pause Controls
* ❤️ User-Friendly Design
* 📱 Responsive Android UI
* 🚀 Fast Performance

## 🛠 Tech Stack

* Java / Kotlin
* Android Studio
* XML
* Gradle
